## ❓常见问题

### 安全吗？

他是安全的，代码完全开源，而且数据内容存在本地的sqlite数据库，无任何网络请求。

#### 开始

```bash
# step1:
git clone https://github.com/shijianzhong/tarui-vue3-handle-note

# step2:
cd tarui-vue3-handle-note

# step3: install deps
pnpm i

# step4:
cargo tauri dev

# step5:
# bundle path: src-tauri/target/release/bundle
cargo tauri build
```
